# Pagina Funzionalità

Questa versione aggiunge la pagina pubblica:
`/?public_page=features`

Contiene:
- hero dedicata alle funzionalità
- moduli CRM, contratti, pagamenti, fatture, lavori, documenti, feedback, staff, SaaS
- flusso operativo
- tabella permessi
- FAQ funzionalità
- CTA finale

La pagina viene renderizzata in iframe isolato, per evitare problemi di HTML mostrato come testo in Streamlit.

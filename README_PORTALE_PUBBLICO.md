# CUBE SaaS - Portale pubblico + pacchetti

Questa versione aggiunge una parte pubblica non amministrativa.

## Parte pubblica
Quando un visitatore entra nel sito senza login vede:

- home page del programma
- spiegazione del gestionale
- vantaggi principali
- pacchetti abbonamento
- tasto Accedi
- tasto Registrati
- scelta pacchetto
- registrazione azienda con 30 giorni gratuiti

## Registrazione
Il cliente:
1. clicca Registrati
2. sceglie il pacchetto
3. compila dati azienda
4. crea il proprio Admin Azienda
5. attiva 30 giorni di prova gratuita

## Super Admin
La dashboard Super Admin non è visibile al pubblico.
Il Super Admin entra solo dalla pagina Login con le credenziali riservate.

Credenziali iniziali:
username: superadmin
password: admin123

## Nota
Il login Super Admin non viene più pubblicizzato nella home o nella schermata pubblica.

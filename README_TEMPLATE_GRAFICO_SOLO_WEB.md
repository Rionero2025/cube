# CUBE Management Contract - versione senza Shopify

Questa versione del progetto NON usa Shopify.

## Cosa è stato fatto
- rimosso ogni riferimento e struttura Shopify/Liquid
- mantenuto solo il concept grafico premium come riferimento visuale
- integrata una landing page completa direttamente nel progetto in `public_site/`
- lasciato il SaaS / portale online separato nella root del progetto

## Contenuto del pacchetto
1. `app.py` e file di deploy del portale SaaS
2. `public_site/` con sito web completo e responsive
3. documentazione e file di supporto

## Obiettivo
Avere un progetto unico dove:
- il portale SaaS resta online
- il sito pubblico usa la stessa identità grafica premium
- non viene usato Shopify in alcun punto

# CUBE SaaS - Super Admin Completo

Questa versione aggiunge il controllo completo Super Admin:

## Nuove funzioni

### Aziende SaaS
- elenco aziende
- creazione manuale azienda
- creazione automatica primo Admin Azienda
- modifica completa anagrafica azienda
- upload/sostituzione logo
- cambio stato: Attivo, In prova, Sospeso, Scaduto, Disattivato
- cambio piano
- gestione utenti/admin aziendali

### Piani
- elenco piani
- crea piano
- modifica piano
- attiva/disattiva piano
- modifica prezzo, limiti e funzioni

### Utenti globali
- vista tutti gli utenti
- modifica utenti di qualsiasi azienda
- cambio ruolo
- cambio stato
- reset password
- cambio tenant/azienda

### Dati SaaS
- vista globale su clienti, contratti, pagamenti, fatture, lavori, documenti, feedback.
